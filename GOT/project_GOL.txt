Edukativní game of life v JS a "publišnuté" pommocí Github

EDU - edukatní stránka
GOL - game of life

Postup
0. Poližit základy
a) Přichystat si (řádně) prostředí pro přípravu programování (založit repozitář -> git init). 
b) Postupovat po "přesně" daných krocích a inkrementálně (-> git add, git commit*). Nikoliv z nuly na sto řádků kódu.
c) Nahrání kódu na Github (git push).
 
1. Naprgrogramovat GOL
a) Podívat na můj GOL v jazyce python a na předchozí projekty užívající canvas API.
b) Přijít s postupem, jak to naprogramuji, abych ho mohl vysvětlit v EDU**. 



*s řádným popiskem účiněného změny
*např. 1. vytvořit kanvas o požadované délce a výšce; 2. vytvořit čtverečky (o požadované délce) 

OTÁZKY_________________________
- Oddělit samotnou GOL a edukativní stránku?
- Udělat GOL se stejnými "fučury" jako má pythonská verze (-> přidat tlačítko na zastavení, ukončení/vymazání, a novou hru)


OBECNÉ POZNÁMKY________________________________
- Nepoužívat příliš familiární tón
- Angličtina
- udělat si pořádek v terminologii
- reakce na změnu rozlišení (velikosti) stránky
- MOžnost přidávat, mazat živé buňky po během odvíjící se hry


FEAČURY________________________
- Nad canvasem hry přidat dvě tlačítka, obě mající dvě funkce závislé na stavu: 
Button A: Start (začne hru se zadou nultou generací); po kliknutí na -> END (or Erase) (ukončí hru a vymaže kanvas)*
Button B: Stop (pozastavý hru v dané generaci); pro kliknutí se změní na -> Continue (hra se odvíjí od současné generace)
- posouvátko udávající rychlost předchodu jedné generace od druhé

* učinit, aby se hra nezačínala bez žádné živé buňky
